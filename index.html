ICD360S e.V. Mail — Flatpak Repository

Auto-Update für den Mail-Client. Einmal hinzufügen, danach läuft flatpak update automatisch beim nächsten Release.

Installation

flatpak remote-add --user --if-not-exists icd360s-mail \
  https://icd360s-e-v.github.io/mail/icd360s-mail.flatpakrepo
flatpak install --user icd360s-mail de.icd360s.mailclient

Update

flatpak update --user
